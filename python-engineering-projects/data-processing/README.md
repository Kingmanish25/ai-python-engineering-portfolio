# Data Processing Tools

This directory contains Python projects focused on **data cleaning, transformation, processing, and analysis pipelines**.
These tools demonstrate practical data engineering workflows using Python for handling structured and semi-structured datasets.

The projects illustrate common data processing tasks such as:

* Cleaning and preparing datasets
* Removing duplicate records
* Generating dataset statistics
* Transforming JSON data
* Processing large files efficiently
* Parsing log files

These tools reflect real-world tasks commonly performed in **data engineering, analytics pipelines, and machine learning data preparation workflows**.

---

# Projects Included

## CSV Data Cleaner

A utility that cleans CSV datasets by removing missing values.

**Key Concepts**

* Data cleaning
* Handling missing values
* Data preprocessing

---

## Data Deduplication Tool

Removes duplicate records from datasets.

**Key Concepts**

* Data deduplication
* Data quality improvement
* Efficient dataset management

---

## Dataset Statistics Analyzer

Generates descriptive statistics for datasets.

**Key Concepts**

* Statistical analysis
* Dataset profiling
* Exploratory data analysis

---

## JSON Data Transformer

Transforms Python objects into structured JSON data.

**Key Concepts**

* JSON processing
* Data serialization
* Data format transformation

---

## Large File Processor

Processes large files efficiently without loading the entire dataset into memory.

**Key Concepts**

* Streaming file processing
* Memory-efficient data handling
* Large dataset workflows

---

## Log File Parser

Parses system log files and extracts important entries such as error messages.

**Key Concepts**

* Log analysis
* Error detection
* Monitoring data pipelines
