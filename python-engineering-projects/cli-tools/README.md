# Python CLI Tools

This directory contains a collection of **Python command-line interface (CLI) utilities** designed to perform common development, automation, and productivity tasks directly from the terminal.

These tools demonstrate practical **Python engineering skills**, including command-line argument parsing, file processing, automation utilities, and network-based tools.

---

## Overview

Command-line tools are widely used in **software engineering, DevOps, and automation workflows**.
The projects in this directory demonstrate how Python can be used to build lightweight CLI utilities for everyday tasks.

This collection includes tools for:

* Data format conversion
* Log file analysis
* Password generation
* Task management
* Network utilities
* Weather information retrieval

Each project is designed as a **standalone CLI tool** with its own documentation and dependencies.

---

## Projects Included

### 1. JSON CSV Converter CLI

A command-line tool that converts JSON data into CSV format.

**Key Features**

* JSON to CSV data conversion
* CLI-based file input and output
* Lightweight data processing utility

---

### 2. Log Analyzer CLI

A tool that analyzes log files and counts occurrences of error entries.

**Key Features**

* Log file analysis
* Error detection
* Simple log monitoring utility

---

### 3. Password Generator CLI

A command-line utility for generating secure random passwords.

**Key Features**

* Random password generation
* Configurable password length
* Includes letters, digits, and special characters

---

### 4. Todo Manager CLI

A CLI-based task management tool that allows users to add and view tasks.

**Key Features**

* Add and list tasks
* Persistent storage using JSON
* Lightweight productivity tool

---

### 5. URL Status Checker CLI

A network utility that checks the HTTP status code of a website.

**Key Features**

* HTTP request status checking
* Network diagnostics utility
* CLI-based URL input

---

### 6. Weather CLI Tool

A command-line tool that retrieves real-time weather information for a city.

**Key Features**

* Fetch weather data from `wttr.in`
* CLI-based city input
* Lightweight weather utility
