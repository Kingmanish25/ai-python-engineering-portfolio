# Python Automation Tools

This directory contains a collection of **Python automation utilities** designed to automate repetitive tasks, monitor systems, and improve productivity through scripting.

These tools demonstrate practical **Python engineering skills**, including file automation, system monitoring, task scheduling, and email automation.

---

## Overview

Automation is one of the most powerful applications of Python in real-world environments.
The projects in this folder demonstrate how Python can be used to automate common workflows and operational tasks.

This collection includes utilities for:

* File and folder automation
* System monitoring
* Task scheduling
* Email automation
* Backup and synchronization tools

Each project is structured as an independent automation tool with its own documentation and dependencies.

---

## Projects Included

### 1. Automated Backup System

A Python automation script that creates **timestamped backups** of a directory.

**Key Features**

* Automatic folder backup
* Timestamped backup directories
* Simple data protection utility

---

### 2. Email Automation Sender

A Python tool for **sending automated emails using SMTP**.

**Key Features**

* SMTP email automation
* Secure TLS connection
* CLI-based email sending

---

### 3. File Organizer

Automatically organizes files into folders based on **file extensions**.

**Key Features**

* Sort files by extension
* Automatically create folders
* Useful for cleaning download directories

---

### 4. Folder Sync Tool

Synchronizes files between two folders.

**Key Features**

* File synchronization
* Copy updated files
* Simple backup and replication utility

---

### 5. Scheduled Task Automation

A lightweight automation script that runs tasks at **scheduled intervals**.

**Key Features**

* Periodic task execution
* Automation scheduling
* Continuous background execution

---

### 6. System Resource Monitor

A monitoring utility that tracks **CPU and memory usage**.

**Key Features**

* CPU usage monitoring
* Memory usage monitoring
* Lightweight system monitoring tool
