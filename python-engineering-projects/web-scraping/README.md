# Web Scraping Projects

This directory contains Python projects focused on **web data extraction, automated data collection, and web scraping pipelines**.

These tools demonstrate practical techniques used for collecting structured data from websites, APIs, and online platforms using Python.

The projects highlight how web scraping can be used for **data gathering, monitoring online content, and building automated data pipelines**.

---

# Projects Included

## Blog Content Scraper

Extracts blog post titles from a website.

**Key Concepts**

* HTML parsing
* Content extraction
* Automated blog monitoring

---

## Crypto Price Scraper

Retrieves real-time cryptocurrency prices using a public API.

**Key Concepts**

* API data retrieval
* JSON parsing
* Cryptocurrency price monitoring

---

## E-commerce Price Tracker

Scrapes product titles and prices from an online store.

**Key Concepts**

* Product data extraction
* Price monitoring
* E-commerce data analysis

---

## Job Listing Scraper

Extracts job listings including title, company, and location.

**Key Concepts**

* Job market data collection
* Structured web scraping
* Recruitment data pipelines

---

## News Scraper

Scrapes news article titles and links from a news website.

**Key Concepts**

* News aggregation
* Content monitoring
* Web content extraction

---

## Wikipedia Data Extractor

Extracts textual information from Wikipedia articles.

**Key Concepts**

* Knowledge extraction
* Research data collection
* Structured text parsing
